# Radar de Faltas

Protótipo desenvolvido no ecossistema MIM para explorar soluções que reduzam a burocracia relacionada ao controle de frequência escolar.

## Objetivo

Validar hipóteses por meio de pequenos experimentos antes do desenvolvimento de uma solução completa.

## Tecnologias

- Node.js
- Playwright

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Crie o arquivo `.env` a partir do modelo:

```bash
cp .env.example .env
```

3. Execute o projeto:

```bash
npm run login
```
